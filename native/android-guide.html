Devkit2 - Developer Preview

This documentation is for a developer preview of Devkit2. Devkit2 is a complete replacement for (and vast improvement over) Devkit1. However, while the api is fairly stable, anything can change between now and the official 2.0 release, particularly in the `devkit` setup commands.

This documentation is currently a work in progress - if you run into any issues, inconsistencies, or other problems, please file an issue at https://github.com/gameclosure/docs2.

You can find the original devkit1 docs at docs-v1.gameclosure.com.

Android Development Guide

This guide takes you through the entire process of developing for Android devices with the Game Closure DevKit.

Overview

The development cycle can be broken down into four stages, leading up to deployment on Google Play. Initially, all development is done with web tools and emulated devices.

Then the Test App is used to accelerate transition onto mobile devices.

Then Android building is encouraged to test and demo your game, with the help of remote debugging tools.

Finally your app is ready for deployment via Google Play.

Setup

The Game Closure DevKit setup guides will get you up and running as fast as possible.

Test App

With the Test App you can quickly iterate through changes to your game on a mobile device.

Build

When you are ready to start building your game, the build guide will make the process smooth and showcase all of the powerful options available.

Remote Debugging

The integrated remote JavaScript debugging allows you to isolate and annihilate all the bugs.

Troubleshooting

There is a brief troubleshooting guide available here.

Plugins

The Android native code can be modified in a reusable way by using the provided plugin system.
